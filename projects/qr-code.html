QR Code Generator

Description

This self-project involves developing a Node.js application that converts URLs into QR code images, improving user accessibility and interaction. The QR codes can be saved for future reference.

Technologies Used

  • Node.js
  • npm packages: inquirer, qr-image
  • File System (fs) module

Key Features

  • Developed a Node.js application to convert URLs into QR code images, enhancing user accessibility and interaction.
  • Utilized the Node.js fs module for efficient file system operations, ensuring smooth data management and storage handling.
  • Integrated npm packages such as inquirer to gather dynamic user input and qr-image to generate QR codes seamlessly.
  • Implemented the core functionality within an index.js file, ensuring a streamlined and scalable process for optimal performance.
  • Improved user experience by automatically saving the input URL into a my-url.txt file and the corresponding QR code as qr-my.png, maintaining an organized record of all generated QR codes for future reference.

Challenges Faced

One of the main challenges was ensuring proper handling of file I/O operations, especially with large amounts of data. Additionally, integrating dynamic user input efficiently was key to providing a smooth experience.
