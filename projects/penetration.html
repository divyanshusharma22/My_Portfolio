Android Device Pentesting & Automation

Description

This project involves automating penetration testing on Android devices to assess security vulnerabilities. The automation process uses multiple tools such as Bettercap, Aircrack-ng, and Ettercap to perform WiFi penetration testing, network traffic sniffing, and DNS spoofing.

Technologies Used

  • Python
  • Kali Linux
  • Bettercap
  • Aircrack-ng
  • Ettercap

Key Features

  • Engineered a Python script for WiFi penetration testing on Android devices, enhancing automation in security evaluations.
  • Utilized Bettercap for network traffic sniffing on Kali Linux, capturing packets to analyze potential security risks.
  • Employed Aircrack-ng to crack network passwords by setting up a local network for targeted vulnerability testing.
  • Conducted DNS spoofing on the testing device using Ettercap, demonstrating the security impact to a panel of professors.

Challenges Faced

Some of the challenges included configuring the tools properly on Kali Linux and ensuring the penetration tests were non-disruptive to the testing environment. Managing the execution flow of multiple security tools was also critical to automating the process successfully.
