House Price Prediction

Description

This project involves predicting house prices using machine learning algorithms like Linear Regression, Decision Tree, and Random Forest. The model is fine-tuned for higher accuracy and to handle any data inconsistencies.

Technologies Used

  • Linear Regression
  • Decision Tree
  • Random Forest
  • Python
  • Scikit-learn
  • Pandas
  • Seaborn (for data visualization)

Key Steps

  • Engineered a house price prediction model using Linear Regression, DecisionTree, and Random Forest algorithms.
  • Fine-tuned model parameters and used cross-validation techniques, achieving a final RMSE of 3.21 for accuracy.
  • Handled missing values with Simple Imputer, filling them with the median to ensure data consistency and accuracy.
  • Standardized the dataset using StandardScaler and created a robust pipeline for effective data handling and processing.
  • Visualized data correlations with the seaborn library to analyze and understand relationships between various dataset features.

Challenges Faced

One of the main challenges was dealing with missing data and ensuring that the model was not overfitting or underfitting. Data preprocessing and feature engineering played a crucial role in achieving the model's accuracy.
