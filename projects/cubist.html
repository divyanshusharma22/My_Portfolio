Cubist Regression Model

Description

This project involves the implementation of a Cubist regression model using a weight adjustment algorithm to calculate the upper bounds of 10 diverse functions. The model iterates over a set of rules from 1 to 5 to optimize predictions and minimize residual errors. The project aims to explore the application of Cubist regression and improve predictive accuracy through iterative weight adjustments.

Technologies Used

  • R Programming
  • Cubist Library
  • Data Visualization (Plots and Graphs)

Key Features

  • Developed a Cubist regression model to predict the upper bounds of 10 diverse mathematical functions.
  • Implemented an iterative weight adjustment algorithm to enhance prediction accuracy.
  • Evaluated model performance using metrics like MSE (Mean Squared Error) and the percentage of points below the predicted curve.
  • Visualized the performance of the model for each function and rule count, highlighting areas where predictions were below the actual curve.
  • Saved results and plots for each function and rule count, ensuring reproducibility of the experiment.

Challenges Faced

Some challenges included fine-tuning the weight adjustment algorithm and ensuring proper model convergence. The large number of iterations required for each function and rule set posed computational challenges, particularly when generating detailed visualizations for each model.
