Movie Recommender System

Description

This system recommends movies based on collaborative filtering and user preferences. It uses machine learning models to suggest relevant movies to users based on their ratings.

Technologies Used

  • Collaborative Filtering
  • Python
  • Machine Learning
  • Pandas, Scikit-learn

Challenges Faced

One of the main challenges was gathering a large dataset for movie ratings and filtering out noisy data.
